bindings

This directory contains files for your custom bindings. Each binding should be a directory with the binding's name and contain at least an index.js that exports a { init, update } binding object.

Make it a best practice to also include a README.md and test.js for your binding in the same directory.

For example: 


      bindings/
      -- fadeIn/
      ---- index.js
      ---- test.js
      ---- README.md

Bindings in this directory will be loaded automatically
