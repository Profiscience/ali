components

This directory contains files for your components. Components should be a directory with the component's name and contain at least an index.js that exports a component config object, and template.html that has the component template, and optionally a viewModel.js with the component viewModel.

For example: 


      components/
      -- side-nav/
      ---- index.js
      ---- template.html

Component index.js files have the convention 


      export { default as viewModel } from './viewModel.js'
      export { default as template } from './template.html'

Components in this directory will be loaded automatically
