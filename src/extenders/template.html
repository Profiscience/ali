extenders

This directory contains files for your extenders. Extenders should be a directory with the extender's name and contain at least an index.js that exports the extender function.

For example: 


      extenders/
      -- foo/
      ---- index.js
      ---- test.js
      ---- README.md

Make it a best practice to also include a README.md and test.js for your extender in the same directory.

Extenders in this directory will be loaded automatically
