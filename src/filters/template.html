filters

This directory contains files for your knockout-punches filters. Filters should be a directory with the filter's name and contain at least an index.js that exports the filter function.

For example: 


      filters/
      -- fadeIn/
      ---- index.js
      ---- test.js
      ---- README.md

Make it a best practice to also include a README.md and test.js for your filter in the same directory.

Filters in this directory will be loaded automatically
