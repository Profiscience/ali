views

Views are defined in the root of src, with each page having its own directory, and several files such as index.js, template.html, viewModel.js, style.scss, model.js, etc.

The contents exported by index.js depends on the plugins you've defined for your app. By default, the following properties are supported... 


      // bar is another view declaration like this
      import bar from './bar'

      export default {
        title: (ctx) => 'My App',
        component: () => ({
          template: import('./template.html'),
          viewModel: import('./viewModel.js')
        }),
        styles: () => [
          import('./style.scss')
        ],
        query: {
          searchText: '',
          sort: 'alpha'
        },
        data: () => import('./model'),
        with: {},
        routes: {
          '/bar': bar
        }
      }

Check each plugin's documentation for details.
